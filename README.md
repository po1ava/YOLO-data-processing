# YOLO模型训练与图像数据处理
本项目为YOLO数据处理与模型训练提供了多种便捷的工具，包括数据集划分、标签数据更改、图像数据增强、模型格式转换等功能。

## 数据集划分
**data_division. py** 程序提供了数据集划分功能，输入路径即可按照预定的划分比例将图片划分至训练集、验证集、测试集文件夹。
```
example/ （运行前）
├── images/ 
└── labels/ 
```
```
example/ （运行后）
├── train/ 
│   ├── images/  
│   └── labels/  
├── val/ 
│   ├── images/  
│   └── labels/  
└── test/ 
    ├── images/ 
    └── labels/
```
